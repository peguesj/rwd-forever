## [1.0.0] - 2026-03-03

### Added
- Initial release
- ARM64 binary patch for Rewind.app v1.5607 cutoff date bypass
- Native macOS installer and uninstaller app bundles with GUI dialogs
- CLI installer/uninstaller for advanced users
- Verification script (`make verify`)
- Custom DMG with background artwork and icon layout
- Automatic binary backup before patching
- Ad-hoc code re-signing with entitlement preservation
- Post-patch configuration (recording enabled, auto-update disabled)
- Reverse engineering intelligence report (INTEL.md)
- SHA256 checksums for release verification
